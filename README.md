# シンプルToDoアプリ

シンプルでおしゃれなToDoアプリです。タスク管理をシンプルに行えます。

## 機能

- ✅ **タスク追加**: 新しいタスクを入力欄から追加できます
- ✓ **完了チェック**: タスクをクリックして完了状態を切り替え
- 🗑️ **削除**: 不要なタスクを削除できます
- 🔔 **通知**: タスク追加時にブラウザ通知を受け取れます
- 💾 **ローカルストレージ**: タスクがブラウザに保存されます

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **ランタイム**: Node.js 16.8以上

## インストール

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くとアプリが表示されます。

## ビルド

```bash
npm run build
```

## 本番環境で実行

```bash
npm start
```

## Vercelへのデプロイ

### 方法1: GitHub連携

1. プロジェクトをGitHubにプッシュ
2. [Vercel](https://vercel.com)にアクセス
3. 「Import Project」をクリック
4. GitHubリポジトリを選択
5. デプロイボタンをクリック

### 方法2: Vercel CLIでデプロイ

```bash
npm install -g vercel
vercel
```

プロンプトに従って設定するだけでデプロイできます。

## 使い方

1. 下部の入力欄にタスクを入力
2. 「追加」ボタンをクリック
3. タスクの左側の円をクリックして完了状態を切り替え
4. 「削除」ボタンでタスクを削除

初回時に「🔔 通知を有効」をクリックすると、ブラウザ通知が有効になります。
